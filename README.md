# Cognivo — Visual Second Brain Canvas

**Live Demo:** [https://cognivo-wltc.onrender.com](https://cognivo-wltc.onrender.com)

**Demo Video:** [Add your YouTube/Loom link here]

An infinite spatial canvas for productivity and life-mapping. Built for the Devlynix Buildathon (Track 6).

## Tech Stack

- **Framework:** Next.js (App Router)
- **UI:** React, React Flow, Lucide Icons
- **State:** Zustand
- **AI:** Google Gemini 1.5 Flash (via `@google/generative-ai`)

## Features

- **Infinite Canvas** — Drag, zoom, and organize nodes freely
- **Node Types** — Goals, Tasks, and Milestones with custom visuals
- **Smart Brain Dump** — Paste or speak your thoughts; AI auto-sorts them into the Eisenhower Matrix
- **Voice Input** — Speech-to-text for brain dumps (Chrome/Edge)
- **Eisenhower Matrix** — Classify tasks by urgency and importance
- **YouTube Roadmap Extractor** — Paste a video URL, extract a step-by-step roadmap onto the canvas
- **Conflict Detection** — Flags overlapping deadlines and illogical task dependencies
- **Time Reality Check** — Warns if your workload exceeds available time before a deadline
- **Node Detail Editor** — Edit label, description, deadline, priority, mark complete, or delete any node
- **Search & Focus** — Quickly find and jump to any node
- **Persistent Canvas** — All nodes and edges auto-save to localStorage
